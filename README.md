# brain-rej · *chimera*

A self-improving research pipeline for discovering blood-borne mediators of brain rejuvenation.
Integrates plasma proteomics, heterochronic-parabiosis scRNA-seq, and human brain aging data
through an auto-critiquing iteration loop.

> *chimera* — heterochronic parabiosis literally creates a chimera (two organisms sharing one bloodstream); the pipeline asks which receptors transduce that shared signal into rejuvenation or aging of the brain.

## What it does

The loop:
1. Runs an analysis defined by a JSON `Plan`
2. Critiques the result against a checklist of known biomedical-analysis failure modes
3. Auto-fixes what it can; escalates what it can't
4. Regenerates `PLAN.md` and `RESULTS.md` as final-product narrative documents

Currently it analyzes 4 papers ([listed below](#data-sources)) to rank brain receptors by their likely causal role in brain aging.

## Quick start

```bash
git clone https://github.com/liuyongyan/brain-rej
cd brain-rej
pip install -r requirements.txt

# Recreate raw data (~770MB; mostly Ximerakis Suppl. Tables)
bash scripts/download_data.sh

# (Optional but recommended) Enable the LLM-as-biomedical-expert critique layer
export ANTHROPIC_API_KEY=sk-ant-...

# Run the loop
python -m pipeline.loop --fresh

# Browse what happened
python -m pipeline.inspect              # all iterations
python -m pipeline.inspect 1            # detail of iteration 1
python -m pipeline.inspect diff 0 1     # what changed between iterations
```

The pipeline runs without `ANTHROPIC_API_KEY` (the LLM critic gracefully degrades to a single "no_api_key" minor issue), but the expert review is the most valuable part of each iteration — set the key if you can, or use `--no-llm` to suppress the prompt entirely.

After the loop finishes, read **`RESULTS.md`** for findings and **`PLAN.md`** for the methodology.

## Data sources

| Paper | Role |
|---|---|
| [Lehallier 2019 *Nat Med*](https://doi.org/10.1038/s41591-019-0673-2) | Human plasma proteome across lifespan (n=4,263) — the "input signal" |
| [Pálovics 2022 *Nature*](https://doi.org/10.1038/s41586-022-04461-2) | Mouse multi-organ heterochronic-parabiosis scRNA-seq — independent replication |
| [Ximerakis 2023 *Nat Aging*](https://doi.org/10.1038/s43587-023-00373-6) | Mouse brain heterochronic-parabiosis scRNA-seq + CellChat — primary brain source |
| [Jeffries 2025 *Nature*](https://doi.org/10.1038/s41586-025-09435-8) | Human PFC snRNA-seq across lifespan — cross-species validation |

PDFs in `paper/` are included for convenience; the underlying datasets are downloaded by `scripts/download_data.sh` from each publisher's supplementary materials and from public databases (OmniPath, HGNC).

## Extending the pipeline

To add a new failure mode: write a check function in `pipeline/checks.py`, add a fix handler in `pipeline/revise.py` if auto-fixable. See `pipeline/USAGE.md` for examples.

To run on a different question: write a new `Plan` JSON, run `python -m pipeline.loop --plan path/to/your_plan.json`. The analysis engine in `pipeline/analyze.py` is currently scaffolded for parabiosis-style multi-source receptor integration; rewriting it for a different question requires editing that file but the loop/critique/revise machinery is reusable.

## Status
